What is web assembly? and why should you care?

Learn how WebAssembly is bringing near-native performance to the browser, by allowing developers bring their native programming language code to the web.

By Jim Milton | June 27, 2019, 10:00pm PT

It's a new capability for the web. It brings the performance of native applications to web and enables full functionality and security you would see from the entire spectrum of software you can run on the computer. It's not a replacement for javascript it works alongside javascript. It's a low-level assembly-like language with a binary format which is very similar to machine code. Intended mostly as a compilation target. So what it lets you do is to take C++ C or Rust(support coming to more languages) which is then compiled into a Web assembly (bytecode). The bytecode is then run on the client—typically in a web browser—where it’s translated into native machine code and executed at high speed. WebAssembly (abbreviated Wasm) is a binary instruction format for a stack-based virtual machine.

Why should you care?

So what can it do for you???? so lets say you have a software already built up in C, C++, or Rust with already thousands of lines of code. Well with wasm you can run that code and display the program in the browser. Software such as video games and high-performance apps for things like computer-aided design, video and image editing. It brings endless possibilities to the web. Or have a high-speed math function already written in C, and you want to incorporate it into a web application, you could deploy it as a WebAssembly module. The less performance-critical, user-facing parts of the app can remain in JavaScript.

Before with Javascript it will been have required plug-ins to achieve near-native performance in the browser, but with WebAssembly running entirely within the Web Platform, developers can integrate WebAssembly libraries for CPU-intensive calculations (e.g. compression, face detection, physics) into existing web apps that use JavaScript for less intensive work.
