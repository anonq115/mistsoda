Overflow Finder

Outlines every element in the document

CSS


                                * {
                                    outline: 1px solid red;
                                  }
What it does:

add that into the Inspector or to your css stylesheet as a new style to see which element is misbehaving if it isn’t immediately apparent. This works especially well when you’ve got something that appears to be taking up more or less space than you think it should or something is forcing horizontal scrolling but you aren’t sure which element is causing it

Javascript Stackoverflow error automatic search

Javascript error catcher

Javascript


                            try {
                                $('body').addClass('body');
                                } catch (e) {
                                  window.open(
                                    "http://stackoverflow.com/search?q=[js] +" + e.message,
                                   '_blank'
                                  );
                               }
What it does:

when an exception is caught in javascript, you get an object (named e in the code snippet) that contains some information about what happened. In this case, the message property of that object is being used to pass the error message into the stackoverflow search URL.

Css Custom Variables

HTML

   <p class="custom-variables">CSS is awesome!</p>

CSS

                                 
                                        :root {
                                            /* Place variables within here to use the variables globally. */

                                            --randomcolor: #da7800; /*this is how you declare your variable. you start with two minus signs then you give a name after that a property value. but of course you can seperate the words with a minus sign like so --random-color looks much better */ you can also place them inside another elements styles locally like shown below.
                                            
                                            --some-keyword: italic;
                                            --some-size: 1.25em;
                                            --some-complex-value: 1px 1px 2px whitesmoke, 0 0 1em slategray, 0 0 0.2em slategray;
                                        }
                                          .custom-variables {
                                           --this-is-a-local-variable:red;
                                            color:var(--this-is-a-local-variable);/*this is the markup for using your css variables*/
                                            color: var(--random-color);
                                            font-size: var(--some-size);
                                            font-style: var(--some-keyword);
                                            text-shadow: var(--some-complex-value);
                                          }
What it does:

Make and use custom variables throughout your project which can also be accessed by javascript: // get variable from inline style element.style.getPropertyValue("—-my-var"); // set variable on inline style element.style.setProperty("--my-var", jsVar + 4);
